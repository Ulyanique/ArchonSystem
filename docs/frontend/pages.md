## Приоритеты рефакторинга

### P0 — Критичные (>1000 строк)
- **BookEditorPage.tsx** (2182) → разбить на 8-12 суб-компонентов
- **SpacePage.tsx** (1249) → отделить 3D view, controls, overlay UI
- **ChatPage.tsx** (991) → страница должна оркестрировать, а не владеть логикой

### P1 — Высокие (500-999 строк)
- **UniverseViewPage.tsx** (943) → извлечь reader components
- **DevelopUniversePage.tsx** (864) → разбить по AI-генерации табам
- **TimelinePage.tsx** (853) → отделить визуализацию от формы
- **SettingsPage.tsx** (815) → разбить по категориям настроек
- **CharactersPage.tsx** (678) → делегировать компонентам

### P2 — Умеренные (300-499 строк)
- 13 страниц в диапазоне 300-555 строк — умеренное разделение
