# LLM-провайдеры и генерация изображений

ARCHON поддерживает 4 LLM-провайдера с автоматическим fallback и 4 провайдера для генерации изображений.

## Архитектура

```
┌─────────────────────┐
│    LLMService       │  Единый интерфейс
└──────────┬──────────┘
           │
    ┌──────┼──────────────┬──────────────┐
    ▼      ▼              ▼              ▼
 Ollama  DeepSeek    OpenRouter     RouterAI
(local)  (cloud)    (aggregator)   (alt)
```

## Текстовые LLM-провайдеры

### 1. Ollama (локальный)

| Параметр | Значение |
|----------|----------|
| URL | `http://localhost:11434` (настраивается) |
| Ключ | Не требуется |
| Модели | Любые, доступные в Ollama |
| Рекомендуемые | `qwen2.5:14b`, `llama3.1:8b`, `mistral:7b` |

**Преимущества:**
- Полная приватность (данные не уходят с машины)
- Нет лимитов API
- Бесплатно

**Ограничения:**
- Зависит от RAM (32 ГБ RAM ≈ модели до ~14B с хорошим контекстом)
- Медленнее облачных провайдеров
- Требует установленный Ollama

**Настройка контекста:**
- Ollama по умолчанию: 2048 токенов
- ARCHON передаёт `num_ctx` (4096–8192+)
- Приложение управляет историей сообщений автоматически

### 2. DeepSeek (облачный)

| Параметр | Значение |
|----------|----------|
| URL | API DeepSeek |
| Ключ | `DEEPSEEK_API_KEY` |
| Модели | `DEEPSEEK_MODEL` (напр. `deepseek-chat`) |

**Преимущества:**
- Высокое качество ответов
- Большой контекст
- Не зависит от локальных ресурсов

**Ограничения:**
- Платный API
- Лимиты запросов
- Данные уходят на сторонний сервер

### 3. OpenRouter (агрегатор)

| Параметр | Значение |
|----------|----------|
| URL | API OpenRouter |
| Ключ | `OPENROUTER_API_KEY` |
| Модели | Любые через OpenRouter (Gemini, Claude, Llama...) |

**Преимущества:**
- Доступ к множеству моделей через один API
- Гибкость выбора
- Хорошее качество

**Ограничения:**
- Платный API
- Дополнительный посредник

### 4. RouterAI (альтернативный)

| Параметр | Значение |
|----------|----------|
| URL | API RouterAI |
| Ключ | `ROUTERAI_API_KEY` |
| Модели | `ROUTERAI_MODEL` |

## Переключение провайдеров

Провайдер выбирается в **глобальных настройках** (`/settings`) или через API:

```
PUT /api/system/settings
{
  "llm_provider": "openrouter",
  "openrouter_model": "anthropic/claude-sonnet-4-20250514",
  "openrouter_api_key": "sk-or-..."
}
```

При недоступности провайдера система автоматически fallback'ится на следующий.

## Генерация изображений

ARCHON поддерживает 4 провайдера для генерации изображений (портреты, концепт-арты, иллюстрации локаций).

### 1. OpenRouter (по умолчанию)

| Параметр | Значение |
|----------|----------|
| Модель | `OPENROUTER_IMAGE_MODEL` (напр. `google/gemini-2.5-flash-image`) |
| Ключ | `OPENROUTER_API_KEY` (тот же, что для текста) |

### 2. Cloudflare Workers AI

| Параметр | Значение |
|----------|----------|
| Ключ | `CLOUDFLARE_API_KEY` |
| Модели | Stable Diffusion через Workers AI |

### 3. Pixazo

| Параметр | Значение |
|----------|----------|
| Ключ | `PIXAZO_API_KEY` |
| Модели | `flux-schnell` (быстрая), `flux-pro` (качественная) |
| Image-to-image | Поддерживается (Flux 2 Pro) |

### 4. Whisk (Google Labs)

| Параметр | Значение |
|----------|----------|
| Ключ | Cookie-аутентификация |
| Модель | Google Whisk |

## Применение изображений

| Тип | Где используется |
|-----|------------------|
| **Портреты персонажей** | Страница персонажей, чат, карточки |
| **Изображения локаций** | Страница локаций |
| **Концепт-арты** | Галерея концепт-артов, pinterest-сетка |
| **Обложки вселенных** | Список вселенных |

## Промпты для генерации

Промпты генерируются автоматически из описания сущности:
- **Персонаж**: внешность + стиль + освещение
- **Локация**: описание + атмосфера + ракурс

Шаблон промпта настраивается в `system_settings.image_prompt_template`.

## Конфигурация (.env)

```env
# Ollama
OLLAMA_BASE_URL=http://localhost:11434
OLLAMA_MODEL=qwen2.5:14b

# DeepSeek
DEEPSEEK_API_KEY=sk-...
DEEPSEEK_MODEL=deepseek-chat

# OpenRouter
OPENROUTER_API_KEY=sk-or-...
OPENROUTER_MODEL=anthropic/claude-sonnet-4-20250514
OPENROUTER_IMAGE_MODEL=google/gemini-2.5-flash-image

# RouterAI
ROUTERAI_API_KEY=...
ROUTERAI_MODEL=...

# Выбор провайдера по умолчанию
DEFAULT_LLM_PROVIDER=ollama
```

## Аппаратные рекомендации

| RAM | Рекомендуемые модели |
|-----|---------------------|
| 16 ГБ | 7B–8B (`llama3.1:8b`, `qwen2.5:7b`) |
| 32 ГБ | 14B (`qwen2.5:14b`, `mistral-nemo:12b`) |
| 64+ ГБ | 32B+ (`qwen2.5:32b`, `deepseek-v3`) |

**Контекст (`num_ctx`):**
- 4096 — минимум для комфортной работы
- 8192 — рекомендуется
- 16384+ — для длинных глав (требует много RAM)
