# Фаза 2: UX и полировка

> Производительность, доступность, пустые состояния, мобильная адаптация. Только после **Фазы 1**.

---

## Модуль 2.1: Производительность

### 2.1.1: Виртуализация списков
- Установить `@tanstack/react-virtual` или `react-window`
- Виртуализация больших списков: персонажи, главы, заметки, концепт-арты
- Рендерить только видимые элементы

### 2.1.2: Оптимизация ререндеров
- `React.memo` для компонентов, которые часто ререндерятся
- `useMemo` для тяжёлых вычислений (граф, фильтрация)
- `useCallback` для функций, передаваемых в дочерние компоненты

### 2.1.3: Image lazy loading
- `loading="lazy"` для всех изображений
- Placeholder (blur/skeleton) во время загрузки
- Progressive JPEG для портретов

### 2.1.4: Debounce поиска и фильтров
- Debounce 300ms для поиска
- Debounce для фильтрации списков
- Throttle для scroll-событий

## Модуль 2.3: Доступность (a11y)

### 2.3.1: ARIA-атрибуты
- `aria-label` для кнопок без текста
- `aria-describedby` для форм
- `role` для landmark-элементов
- `aria-live` для динамических обновлений

### 2.3.2: Keyboard navigation
- Tab-навигация по всем интерактивным элементам
- Горячие клавиши (Ctrl+K, Ctrl+S, Escape)
- Focus trap в модалах
- Skip links

### 2.3.3: Screen reader
- Семантический HTML (`<nav>`, `<main>`, `<aside>`)
- Альтернативный текст для изображений
- Заголовки в правильном порядке (h1 → h2 → h3)

### 2.3.4: Focus management
- Автофокус на первом поле модала
- Возврат фокуса после закрытия модала
- Visible focus indicator

### 2.4.1: Responsive дизайн
- Все страницы адаптируются к ширине < 768px
- Мобильное меню (hamburger)

### 2.4.2: Touch-жесты

### 2.4.3: Мобильный редактор
- Markdown-редактор с мобильной клавиатурой
- Компактный тулбар

## Модуль 2.5: Уведомления и обратная связь

### 2.5.1: Toast для всех операций
- Успех: зелёный toast
- Ошибка: красный toast
- Информация: синий toast
- Предупреждение: жёлтый toast

### 2.5.2: Прогресс-бары для долгих операций
- AI-генерация: прогресс с процентом
- RAG-переиндексация: прогресс
- Загрузка файлов: прогресс
- Экспорт: индикатор

### 2.5.3: Контекст-индикатор в чате
- Сводка: «В контексте: N персонажей, RAG: Y фрагментов»
- Визуальный индикатор загруженного контекста
