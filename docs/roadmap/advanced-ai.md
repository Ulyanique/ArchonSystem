# Фаза 4: Продвинутые AI

> Новые AI-функции после полной стабилизации базы. Только после **Фазы 3**.

---

## Модуль 4.1: AI-ассистент

### 4.1.1: Worldbuilding-помощник
- Анализ полноты мира: что есть, чего не хватает
- Предложения по развитию вселенной
- Обнаружение «белых пятен» (неописанные аспекты)

### 4.1.2: Обнаружение противоречий
- Персонаж: характер vs действия в главах
- Локация: описание vs упоминания в тексте
- Таймлайн: хронологические нестыковки
- Технологии: совместимость с сеттингом

### 4.1.3: Автоматические связи
- Предложение связей между сущностями на основе контекста
- «Персонаж X упомянут в главе Y, но нет связи — создать?»
- «Локация Z описана, но не привязана ни к чему»

---

## Модуль 4.2: AI-редактор

### 4.2.1: Редактирование стиля
- Изменение тона (мрачнее / светлее)
- Изменение темпа (динамичнее / медленнее)
- Изменение POV (от первого лица / от третьего)

### 4.2.2: Проверка качества
- Грамматика и орфография
- Повторы слов и конструкций
- «Показать, не рассказать» (show don't tell)
- Диалоги: звучат ли естественно

### 4.2.3: Работа с текстом
- Сокращение текста без потери смысла
- Расширение сцены деталями
- Переписывание конкретного абзаца

### 4.2.4: Перевод
- Перевод глав на другие языки
- Сохранение стиля при переводе
- Перевод описаний сущностей

---

## Модуль 4.3: Multi-character chat

### 4.3.1: Чат с несколькими персонажами
- Выбор 2+ персонажей для диалога
- AI автоматически переключает роли
- Индикатор «кто говорит»

### 4.3.2: Групповые диалоги
- Моделирование сцен с несколькими персонажами
- Каждый персонаж сохраняет характер и знания
- Возможность задать «сцену» и наблюдать диалог

### 4.3.3: Voice
- Text-to-Speech для ответов (разные голоса)
- Speech-to-Text для ввода
- Интеграция с Web Speech API или внешними сервисами

---

## Модуль 4.4: AI для работы с изображениями

### 4.4.1: Consistent character art
- Генерация изображений одного персонажа в разных позах
- Сохранение стиля между генерациями
- Reference image для консистентности

### 4.4.2: Scene illustration
- Генерация иллюстраций к конкретным сценам
- Image-to-image на основе описания сцены

### 4.4.3: Style transfer
- Применение единого стиля ко всем изображениям вселенной
- Выбор художественного стиля (anime, realistic, watercolor...)
