# Дорожная карта развития ARCHON

Модульная дорожная карта, организованная по фазам — от стабилизации к продвинутым функциям.

## Принцип приоритетов

**Сначала качество, потом новые фичи.** Каждая фаза должна быть завершена перед переходом к следующей.

## Фазы

| Фаза | Название | Статус | Описание |
|------|----------|--------|----------|
| [**0. Стабилизация**](stabilization.md) | 🔄 В процессе | Доведение всех существующих функций до рабочего состояния |
| [**1. Модульность кода**](modularity.md) | ⏳ Ожидает | Рефакторинг монолитных файлов, разбиение типов |
| [**2. UX и полировка**](ux-polish.md) | ⏳ Ожидает | Производительность, доступность, пустые состояния |
| [**3. Тестирование**](testing.md) | ⏳ Ожидает | Unit, integration, E2E тесты |
| [**4. Продвинутые AI**](advanced-ai.md) | ⏳ Ожидает | Новые AI-функции после стабилизации базы |
| [**5. Масштабирование**](scaling.md) | ⏳ Ожидает | Docker, PostgreSQL, мультипользовательность |
