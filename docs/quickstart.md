# Quickstart — AgentSystem для ArchonSystem

## Что это

Саморазвивающаяся система из двух корневых агентов:

- **meta-agent** — управляет жизненным циклом агентов (создаёт, модифицирует, удаляет)
- **universal-agent** — исполняет задачи, выбирая роль на основе приоритетов

## Структура

```
agents/              — промпты агентов
  meta-agent.md      — мета-агент
  universal-agent.md — универсальный агент
state/               — состояние системы (MOC + файлы агентов)
backlog/             — задачи (MOC + файлы агентов)
reports/             — отчёты агентов + summary.md
MANIFEST.md          — ДНК проекта
```

## Как запустить

1. Прочитай `MANIFEST.md` — пойми цель
2. Запусти meta-agent коротким промптом:

```
Прочитай agents/meta-agent.md и выполни инструкции в нём.
```

3. После того как meta-agent создаст роли, запусти universal-agent:

```
Прочитай agents/universal-agent.md и выполни инструкции в нём.
```

## Важно

- **Не запускай два universal-agent одновременно** — это вызовет конфликты в файлах проекта
- meta-agent и universal-agent можно запускать параллельно (они пишут в разные файлы)
- Все отчёты — в `reports/`, сжатая история — в `reports/summary.md`
- Каждый агент пишет только в свои файлы состояния и бэклога
