# ARCHON

> **Система управления вселенными для писателей-фантатов**

ARCHON — это fullstack-приложение для worldbuilding'а: создания и управления вымышленными вселенными, персонажами, локациями, главами, таймлайнами, фракциями, космическими объектами и многим другим. Глубокая интеграция с LLM (Ollama, OpenRouter, DeepSeek, RouterAI) позволяет генерировать контент, анализировать согласованность, писать текст глав и вести интеллектуальный чат с персонажами через RAG.

## 🤖 Агентная система

Проект управляется саморазвивающейся системой из двух корневых агентов:

- **meta-agent** — управляет жизненным циклом агентов, проводит аудит эффективности, обнаруживает стагнацию
- **universal-agent** — исполняет задачи, выбирая роль на основе приоритетов и текущего состояния

Подробнее: [docs/quickstart.md](docs/quickstart.md) | ДНК проекта: [MANIFEST.md](MANIFEST.md)

## 🚀 Быстрый старт

### 1. Установка (один скрипт)
```powershell
.\setup.ps1
```

### 2. Запуск
```powershell
.\start.ps1
```

### 3. Остановка
```powershell
.\stop.ps1
```

> **Бэкенд:** http://localhost:8000  
> **Swagger UI:** http://localhost:8000/docs  
> **Фронтенд:** http://localhost:5173

### Ручная установка

<details>
<summary>Бэкенд</summary>

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
# Для воспроизводимой сборки: pip install -r requirements-pinned.txt
cp .env.example .env
uvicorn app.main:app --reload
```
</details>

<details>
<summary>Фронтенд</summary>

```bash
cd frontend
npm install
# Для строгой установки по lockfile: npm ci
npm run dev
```
</details>

## 🤖 LLM-провайдеры

Система поддерживает 4 провайдера с автоматическим fallback:

| Провайдер | Тип | URL / Ключ |
|-----------|-----|-----------|
| **Ollama** | Локальный | `http://localhost:11434` |
| **DeepSeek** | Облачный | `DEEPSEEK_API_KEY` |
| **OpenRouter** | Агрегатор | `OPENROUTER_API_KEY` |
| **RouterAI** | Альтернативный | `ROUTERAI_API_KEY` |

Генерация изображений: **OpenRouter** (default), **Cloudflare Workers AI**, **Pixazo** (Flux Schnell/Pro), **Whisk** (Google Labs).

Подробности: [docs/llm-providers.md](docs/llm-providers.md)
