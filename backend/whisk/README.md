# Whisk (Google Labs) — мост для генерации изображений

Используется пакет [@rohitaryal/whisk-api](https://github.com/rohitaryal/whisk-api) для вызова Whisk (Imagen) через неофициальный API Google Labs.

**Требования:** Node.js 18+, установленные зависимости.

```bash
npm install
```

Cookie Google задаётся в настройках ARCHON (Настройки → Генерация изображений → Whisk → Cookie Google).
